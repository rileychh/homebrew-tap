# Rileychh Tap

## How do I install these formulae?

`brew install rileychh/tap/<formula>`

Or `brew tap rileychh/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rileychh/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
